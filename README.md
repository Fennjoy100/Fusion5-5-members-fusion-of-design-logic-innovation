# 🎬 ThiraiVerse – Movie Ticket Booking Web App

## 📌 Project Overview

ThiraiVerse is a responsive movie ticket booking web application where users can browse movies, view show timings, select seats, and proceed to booking.

This project is built using **HTML, CSS, and JavaScript** with a focus on UI design and user experience.

---

## 🚀 Features

* 🎥 View Now Showing & Upcoming Movies
* 🔍 Search movies
* 🕒 Select show date, time & theatre
* 💺 Interactive seat selection system
* 🔒 Reserved/occupied seat handling (localStorage)
* 💰 Dynamic price calculation
* 📱 Fully responsive design (Mobile, Tablet, Laptop)
* 💳 Checkout flow integration

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox + Grid + Responsive Design)
* JavaScript (DOM Manipulation)
* Local Storage (for seat booking persistence)

## ⚙️ How It Works

1. User selects a movie from homepage  
2. Chooses language, date, time & theatre  
3. Redirects to **Seat Selection Page**  
4. User selects seats  
5. Total price updates dynamically  
6. On checkout:
   - Seats are stored in **localStorage**
   - Prevents duplicate booking
